SELECT:
  document.getElementById("...")                select id
  document.getElementsByClassName("...")        select class, creates nodelist
  document.getElementsByClassName("...")[...]   [] to select just one
  document.getElementsByTagName("...")          select tag, creates nodelist

  document.querySelector("...")                 needs CSS style selectors, #..., selects only the first one
  document.querySelectorAll("...")              needs CSS style selectors, selects all elements, creates nodelist

  var h1 = document.querySelector("h1");        selects h1 and saves it in a variable

MANIPULATION:
  query selector:
    element.querySelector(selector)               return reference to first selector
    element.querySelectorAll(selector)            return nodelist with all selectors

  create element:
    document.createElement(tag, options)          create element, optional options

  append element:
    parentElement.appendChild(childElement)       add childNode as last child of parentNode
    parentElement.insertBefore(newNode, nextNode) add newNode before nextNode in parentNode

  remove element:
    parentElement.removeChild(childElement)       remove child from parentNode, return reference to child

  alter element:
    .style.color = "..."                          add rule
    .style.cssText = "..."                        add multiple rules, css style
    .setAttribute("style", "...")                 add multiple rules

    background-color -> backgroundColor           naming rules for CSS

  edit attribute:
    .setAttribute("name", "value")                change name="value", or add it
    .getAttribute("...")                          return value of attribute
    .removeAttribute("...")                       remove attribute

  class:
    .classList.add("...")                         add class
    .classList.remove("...")                      remove class
    .classList.toggle("...")                      toggle class, prefered

  content:
    .textContent = "..."                          add plain text, prefered
    .innerHTML = "..."                            add html text

default behaviour:
  element.preventDefault()    prevent default behaviour

---

event bubbling and capture:
  bubbling: browser default
    run event on the element, then parent, ..., then on html

  capturing:
    run event on html, then child, ..., then on the element

  element.stopPropagation()     stop event bubbling

event delegation:
  multiple elements with same behaviour: one event on the parent and bubble each child
